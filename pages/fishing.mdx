# Renewed Fishing

Step into the world of Renewed Fishing, with a fully unqiue and configureable fishing experience.









## Features
- Supports Ox_Core, ESX, QBCore, Qbox and NDCore
- Supports mutli jobs such as slrn_groups
- Fully configureable fish loot table.
- Fully translateable
- Supports entity lockdownmode
- Different Fishing Rods, Hooks and Baits to make sure every fisher can have a unique experience
- Completely overhauled system with build in water and depth checks
- Possible of banning certain areas from fishing
- Fishing tournaments with a API to create your own
- Custom fishing net prop thanks to .ep.
