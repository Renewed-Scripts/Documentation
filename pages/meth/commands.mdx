This page covers all the commands available in Renewed Meth.


## createlab
An admin commands that creates a lab at the specific location the admin places it.

```
/createlab [maxUpgrades]
```

- maxUpgrades: `boolean`


## startlab
An admin command that starts the meth cooking without having to do a tedious animation or items needed

```
/startlab
```

## stoplab
An admin command that stops the meth cooking immediately without having it explode in your face.

```
/stoplab
```
