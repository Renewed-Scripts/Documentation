## CreateLab

Creates a new lab

```lua
exports['Renewed-Meth']:RegisterLootTable(coords, maxUpgrades)
```

- coords: `vector4`
- maxUpgrades: `boolean`


```lua
exports['Renewed-Meth']:RegisterLootTable(vector4(0.0, 0.0, 0.0, 0.0), true)
```


## IsPlayerInLab

Checks if a player is in a lab

```lua
exports['Renewed-Meth']:IsPlayerInLab(source)
```

- source: `string`
- return `number` | `nil`

```lua
    local isPlayerInLab = exports['Renewed-Meth']:IsPlayerInLab(source)

    print(isPlayerInLab) -- Prints lab ID if player is in lab or nil if player is not in lab
```
