## Renewed-Hud

Renewed-Hud brings a fresh feel to how a Heads-Up Display should be for servers, while providing enough customization for players!

### Features
- Easy Position Map and Speedometer
- Customization for Icon Placement
- Classing for Less Overhead on Server Performance

### Dependencies
- [ox_lib](<https://github.com/overextended/ox_lib>)
- [Renewed-Lib](<https://github.com/Renewed-Scripts/Renewed-Lib>)
