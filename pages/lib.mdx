
##  Renewed Lib

Renewed Lib is a collection of functions and framework intergration used for Renewed-Scripts, feel free to use it in your own scripts if you feel like or improve anything by doing a PR.

### Features

- Support for ESX, QBCore, and Ox

- Spawn peds that auto despawn when not in range
- Spawn objects that auto despawn when not in range
- Various framework functions to get a player's group, grade, and more

### Framework Specific Feature(s)

- Supports creating stash shops with `ox_inventory`

### Dependencies

- [ox_lib](https://github.com/overextended/ox_lib/releases)
- Optional: [ox_inventory](https://github.com/overextended/ox_inventory/releases)

- ESX or QBCore or Ox

## Installation

- Install `ox_lib` and have it started before `Renewed-Lib`
- Make sure to have qb/esx installed for framework intergration
- Drag and drop the resource into your resources folder and enjoy!
