## Renewed Meth

Renewed Meth offers a comprehensive meth production system. It provides support for multiple frameworks, integrates securely into your server, and offers language localization for global servers.

### Features
    - **Framework Support:** ESX, QBox, QBCore and Ox Core
    - **Secure Netevents:** Prevents cheaters from being able to exploit the system to gain free items.
    - **Entity Lockdown:** Offers support for three levels of security – inactive, relaxed, and strict.
    - **Target Systems:** Integration with [ox_target](https://github.com/overextended/ox_target) and [interact](https://github.com/darktrovx/interact)
    - **Locale Support:** Supports multiple languages for global servers.
    - **Particle Effects:** Has build-in particle effects for a more immersive experience.
    - **Prop Storage:** Spawns props depending what is in the stashes for a more immersive experience.
    - **Configurable:** Allows you to customize the system to your server’s needs.
    - **Server Sync:** Syncs the system states to all clients in the methlab.

### Dependencies
    - [ox_inventory](https://github.com/overextended/ox_inventory)
    - [ox_lib](https://github.com/overextended/ox_lib)
    - [ox_target](https://github.com/overextended/ox_target) or [interact](https://github.com/darktrovx/interact)
    - [Renewed-Lib](https://github.com/Renewed-Scripts/Renewed-Lib)
    - [ep_methprops](https://github.com/PeliporukkaRP/Methlab)
    - QBCore, ESX, OX Core or QBox
