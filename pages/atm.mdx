## Renewed ATM Robbery

Renewed ATM Robbery is a brand new way of experiencing ATM Robberies with server sided entities!
### Features
   - Fully secure, cheaters won't be able to spawn whatever item on your server

   - Configurable props (if you use a different atm props you can change that in the config)

   - Comes with default ATM Props (Thanks to .ep.)

   - Entity lockdown support (all props are spawned server sided)

   - Synced through statebags

   - ox_target or [interact](https://github.com/darktrovx/interact) support

   - multi framework compatible (ESX, QB, QBOX, OX, ND(WHO) CORE)

   - Configureable police notifications

### Dependencies

   - [ox_inventory](https://github.com/overextended/ox_inventory)

   - [ox_lib](https://github.com/overextended/ox_lib)

   - [ox_target](https://github.com/overextended/ox_target) or [interact](https://github.com/darktrovx/interact)
